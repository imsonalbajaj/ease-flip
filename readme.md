# EaseFlip
A fun and interactive emoji tile-matching game built using SwiftUI, SwiftData, MVVM, Lottie animations, and AVFoundation for sound effects.

2. Open in xcode, along with download the dependencies through SPM
3. run in simulator or your device by sign-in with your id
